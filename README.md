EasyFit is a beginner-friendly fitness mobile application designed to help users start and maintain a healthy gym routine. It serves as a personal gym companion by providing simple guidance, workout tracking, and motivation for users who are new to fitness.

---

## Features

- Beginner-friendly workout guidance  
- Simple and clean user interface  
- Workout and routine tracking  
- Designed for gym starters and casual fitness users  
- Offline-friendly core features

- **Framework:** Flutter  
- **Language:** Dart  
- **Platform:** Android / iOS
